# 期末作业题目及要求
## 一、作业题目
### 题目1：城市交通最短路径规划系统（图算法应用）
#### 问题描述：
设“算法城市”有20个交通节点（编号1-20），M条双向道路。每条道路有长度、拥堵系数、红绿灯数量三个属性。现需设计一个综合路径规划系统，满足不同需求：
1. **基础功能**：实现Dijkstra算法，给定起点和终点，计算基于道路长度的最短路径；
2. **进阶功能**：设计加权最短路径算法，权重 = 距离×拥堵系数（每条道路都的拥堵系数≥1） + 红绿灯数量×等待时间
#### 输入格式：
输入城市20个交通节点的基本信息：任意两个交通节点之间双向通道的长度，每条路的拥堵系数，红绿灯个数，假设每个红绿灯的等待时间均为30秒。
#### 输出要求：
- **有两种查询**：
  1. 输入任意两个交通节点，输出这两个节点之间的最短路径。（基础功能，只考虑道路长短）
  2. 输入任意两个交通节点，输出这两个节点之间的最短通行时间。（进阶功能，考虑道路的拥堵和红绿灯的等待时间）
### 题目2：公交线路换乘优化
#### 问题描述：
城市公交网络包含N条线路，每条线路有若干站点（同一站点可能属于多条线路）。乘客从起点站到终点站，可能需要换乘。
#### 要求：
1. 实现最少换乘次数算法（基于BFS广度优先搜索）
2. 实现综合考虑换乘次数和总站数的算法（加权图）

#### 输入格式：
1. 线路数据、起点站、终点站
#### 输出要求：
1. 输入两个站点，给出最优乘车方案（经过的线路和站点序列）

### 题目3：智能停车场空位分配与引导
#### 问题描述：
停车场有M行N列车位，车辆按预约时间到达，有不同的停车时长需求。
#### 要求：
1. 实现贪心算法：按照某种策略（如就近停车）分配车位
2. 实现回溯算法：寻找最优分配使得总移动距离最小
3. 比较两种算法的结果和性能

#### 输入格式：
1. M和N的值（总车位），t[i]表示第i辆车到达时间，p[i],表示第i辆车停车时间，输入一个车辆进场序列。

#### 输出要求：
1. 停车场里某时刻的停车信息。
2. 离开车辆顺序。
